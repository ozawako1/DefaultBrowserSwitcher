# DefaultBrowserSwitcher 配布手順ガイド

## 前提事項

このアプリは `LSSetDefaultHandlerForURLScheme` を使ってデフォルトブラウザを変更するため、
App Sandbox が有効だと動作しない可能性が高い。
そのため、Mac App Store 経由ではなく、Developer ID による直接配布（Notarization）が推奨される。

---

## Step 1: App Sandbox を無効化する

現在 `ENABLE_APP_SANDBOX = YES` になっているが、
`LSSetDefaultHandlerForURLScheme` はサンドボックス環境では制限される。

Xcode で行う操作:
1. プロジェクトナビゲータでターゲット「DefaultBrowserSwitcher」を選択
2. Signing & Capabilities タブを開く
3. App Sandbox の「×」ボタンで削除する

※ App Store配布を希望する場合は、サンドボックス対応の別の方法を検討する必要があるが、
  デフォルトブラウザ変更は基本的にApp Store非対応。

---

## Step 2: Developer ID 証明書を作成する

### 方法A: Apple Developer サイトから手動で作成

1. Apple Developer サイト (developer.apple.com) にログイン
2. Certificates, Identifiers & Profiles に移動
3. 「+」ボタンで新しい証明書を作成
4. Developer ID Application を選択（直接配布用）
5. CSR（Certificate Signing Request）をキーチェーンアクセスから作成してアップロード
6. 証明書をダウンロード → ダブルクリックでキーチェーンに登録

### 方法B: Xcode で自動管理

1. Xcode → Settings → Accounts で Apple ID を追加
2. チーム N635VGVU4X を選択し「Manage Certificates」をクリック
3. 「+」→「Developer ID Application」を作成

---

## Step 3: コード署名設定を確認する

Xcode のプロジェクト設定で以下を確認：

- Signing Style: Automatic（現在の設定のまま）
- Team: 開発チーム N635VGVU4X（設定済み）
- Bundle Identifier: jp.co.ozawako1.DefaultBrowserSwitcher（設定済み）
- Hardened Runtime: YES（設定済み - Notarization に必須）

---

## Step 4: アプリアイコンを設定する（任意だが推奨）

現在 AppIcon.appiconset は空の状態。配布前にアイコンを設定する：

1. 1024×1024px の PNG アイコン画像を用意
2. Xcode の Assets.xcassets → AppIcon にドラッグ＆ドロップ

---

## Step 5: Archive ビルドを作成する

1. Xcode のスキーム選択で「Any Mac」を選択（シミュレータではなく実機ビルド）
2. メニュー → Product → Archive を実行
3. ビルドが成功すると Organizer ウィンドウが自動で開く

---

## Step 6: Notarize して書き出す

Organizer ウィンドウで：

1. 作成した Archive を選択
2. 「Distribute App」ボタンをクリック
3. 「Direct Distribution」を選択
4. 配布方法として「Developer ID」を選択
5. 「Upload」を選択（Apple の notarization サーバーに送信される）
6. 署名オプションは Automatically manage signing のまま「Next」
7. 「Upload」をクリック

Apple のサーバーで Notarization が実行される（通常数分〜15分程度）。

8. 完了通知が来たら、Organizer で「Export Notarized App」をクリック
9. 保存先を選択 → .app ファイルが書き出される

---

## Step 7: DMG で配布する（推奨）

ユーザーに配布する場合、.app をそのまま渡すのではなく DMG にまとめるのが一般的。

### シンプルな DMG 作成

```bash
hdiutil create -volname "DefaultBrowserSwitcher" \
  -srcfolder /path/to/exported/DefaultBrowserSwitcher.app \
  -ov -format UDZO \
  DefaultBrowserSwitcher.dmg
```

### 凝った DMG を作りたい場合

背景画像や Applications フォルダへのエイリアス付きの DMG を作る場合は、
create-dmg (https://github.com/create-dmg/create-dmg) などのツールが便利。

---

## Step 8: 配布する

作成した DMG を以下の方法で配布できる：
- GitHub Releases に添付
- 自分の Web サイトでダウンロード提供
- その他のファイル配布サービス
